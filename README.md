# 📊 Mi App UTM & Finanzas Personales

Una aplicación web moderna, privada y ligera para llevar el control de tus finanzas personales, diseñada especialmente para Chile.

## 🚀 Acceso Directo
Puedes usar la aplicación directamente aquí:
**[https://kudawasama.github.io/mi-app-utm/](https://kudawasama.github.io/mi-app-utm/)**

## ✨ Características Principales
- **Indicadores en Tiempo Real**: Carga automática de UTM, UF y Dólar (vía mindicador.cl).
- **Privacidad Total**: Tus datos se guardan localmente en tu navegador. Nadie más puede ver tus gastos.
- **PWA Ready**: Instálala en tu teléfono como una aplicación nativa desde el navegador.
- **Gestión Completa**:
  - Registro de Ingresos y Gastos con categorías.
  - Gestión de Gastos Recurrentes (se añaden automáticamente cada mes).
  - **Registro Relámpago (⚡)**: Optimizado para uso rápido en móvil.
  - Edición y eliminación de registros.
- **Visualización**: Gráficos dinámicos para distribución de gastos y evolución de la UTM.
- **Respaldo**: Importación y exportación de datos en formato JSON.

## 📱 Cómo "instalar" en tu móvil
1. Abre el enlace en tu navegador (Safari en iPhone o Chrome en Android).
2. Selecciona **"Compartir"** o el menú de ajustes (⋮).
3. Toca en **"Agregar a la pantalla de inicio"**.
4. ¡Listo! Tendrás el icono en tu escritorio y funcionará a pantalla completa.

## 🛠️ Tecnologías
- HTML5 / CSS3 (Glassmorphism design)
- Vanilla JavaScript
- Chart.js
- Service Workers (Soporte Offline)
